# Final Projects FlipRobo

Collection of Jupyter Notebook machine learning projects covering regression, classification, supervised learning workflows, and model evaluation.

These projects demonstrate data preparation, exploratory analysis, model training, and evaluation across several real-world-style datasets.

## Projects Included

| Project | Focus |
| --- | --- |
| Banking Marketing | Customer response prediction for marketing campaigns |
| Titanic Project | Survival prediction using classification models |
| Glass Classification | Multi-class glass type prediction |
| Happiness Project | World happiness analysis and predictive modeling |
| Medical Cost Personal Insurance | Insurance cost prediction using regression models |
| Restaurant Food Cost | Food cost prediction and analysis |
| Student Grades Prediction | Academic performance prediction |
| Wine Property Project | Wine quality and physicochemical property analysis |

## Techniques Used

- Exploratory data analysis
- Missing-value and outlier handling
- Feature encoding and scaling
- Regression and classification modeling
- Gradient boosting and support vector machines
- Model export with `joblib`

## Tech Stack

Python, Jupyter Notebook, pandas, NumPy, scikit-learn, matplotlib, seaborn, joblib

## How To Use

```powershell
python -m venv .venv
.\.venv\Scripts\activate
pip install pandas numpy scikit-learn matplotlib seaborn jupyter joblib
jupyter notebook
```

Open the relevant project folder and run the notebook for that analysis.
